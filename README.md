# Digital Asset Publishing Kit

A collection of templates and guides for packaging and publishing digital assets on marketplaces like itch.io, Gumroad, or asset stores.

## Features

- **README Template** — Standard format for documenting your asset
- **License Guide** — Help buyers understand usage rights
- **Store Copy Template** — Marketing copy structure for listings
- **Versioning Guide** — How to version and update your assets

## Contents

| File | Purpose |
|------|---------|
| `README_TEMPLATE.md` | Fill-in-the-blank README for your asset |
| `LICENSE_GUIDE.md` | Guidance on choosing and writing licenses |
| `STORE_COPY_TEMPLATE.md` | Structure for marketplace descriptions |
| `VERSIONING.md` | Asset versioning conventions |

## Usage

1. Copy the templates you need into your asset project
2. Fill in the bracketed placeholders
3. Customize to fit your specific asset
